The pattern here is to identify the start of a sentence, any non-trigger/non-punctuation text if it exists, any of the trigger expressions, more non-trigger text if it exists, and finally the ending punctuation.

This has its shortcomings but it will do a quick and dirty job of tokenizing and finding trigger patterns all in one step.
*/

//first we have the sentence tokenization regex patterns
String sentStart = "[A-Z]";
String nonPunc = "[^.!?]*";
String sentEnd = "[.!?]";


//then there are the temporal triggers
String month = "((January|February|March|April|May|June|July|August|September|October|November|December)(\\b))";
String monthshort = "((Jan|Feb|Mar|Apr|May|Jun|Jul|Aug|Sept|Oct|Nov|Dec)(\\b))";
String descriptive = "(((|n)ext|(|l)ast|(|t)his|previous)\\s(year|quarter|month)(\\b))";
String yearnum = "(((|i)n|(|d)uring|(|o)ver|(|b)y)\\s((19)|(20))([0-9]{2}))";
String seasons = "(((|i)n|(|d)uring|(|o)ver|(|b)y)\\s(the)\\s((S|s)ummer|(W|w)inter|(F|f)all|(S|s)pring))";
String descSeasons = "(((|n)ext|(|l)ast|(|t)his|previous)\\s((S|s)ummer|(W|w)inter|(F|f)all|(S|s)pring))";
			
//and the compiler
Pattern p = Pattern.compile(sentStart+
					nonPunc+"("+
					month+"|"+
					monthshort+"|"+
					seasons+"|"+
					yearnum+"|"+
					descriptive+"|"+
					descSeasons+")"+
					nonPunc+
					sentEnd);

//Then I used a matcher to find the pattern in a given string and a boolean to represent whether the pattern is present.
